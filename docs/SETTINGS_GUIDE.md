# 個人設定のしかた（初期設定ガイド）

Kitchen App をもっと便利にもっと安全に使うための「３つのお守り設定」についてご紹介します。難しそうに見えても大丈夫、一つずつ順番に進めれば誰でも設定できます！

---

## 1. Google アカウント連携（データのお守り）

「スマホを買い替えたらレシピがごそっと消えちゃった…」という悲しい事故を防ぐための、いちばん大切な設定です。あなたがお使いの Google ドライブの隅っこに、アプリのデータをまるごと自動で保存（バックアップ）してくれます。

### どうやって設定するの？
1. アプリの **「設定」 > 「アカウント」** を開きます。
2. もし `VITE_GOOGLE_CLIENT_ID` という環境変数が設定されていれば、そこに **「Googleでログイン」** というボタンが表示されます。（※表示されていない場合は、アプリの配布元やご家族の管理者に「Googleログインを有効にしてほしい」とお伝えください。）
3. ボタンを押して、普段お使いの Google アカウントを選ぶだけ！
4. これで完了です。レシピを追加したり献立を変えたりするたびに、自動で安全にバックアップされるようになります。

---

## 2. Google カレンダー連携（家族で共有するためのお守り）

Kitchen App で作った「今週の献立表」や「お買い物リスト」を、スマートフォンのカレンダーアプリに予定として書き込むことができます。ご家族とカレンダーを共有していれば、「今日の夜ご飯なに？」と聞かれる前に自動でお知らせできます！

### どうやって設定するの？
1. まず上記の **Google アカウント連携** を終わらせておきます。
2. アプリの **「設定」 > 「カレンダー」** を開きます。
3. カレンダーへの書き込み許可（スイッチ）をオンにします。
4. **「どのカレンダーに書き込むか」** を選びます（「家族用カレンダー」などの共有カレンダーを選ぶのがおすすめです）。
5. あとは、週間献立の画面で「カレンダー登録」ボタンを押すだけ！ 指定したカレンダーに献立予定と買い物リストがバッチリ登録されます。

---

## 3. Gemini API キー（AIのちからを借りるためのお守り）

「冷蔵庫の残り物から今日の献立を考えてほしい…」「ネットで見つけたレシピを全自動で読み込んでほしい…」そんなときに活躍するのが最新のAI機能です。あなた専用の「合鍵（APIキー）」を取得してアプリに教えることで、AIがフルパワーを発揮してくれます。

### どうやって設定するの？
1. Google の AI向けサイト [Google AI Studio](https://aistudio.google.com/app/apikey) にアクセスします（Googleアカウントでログインしてください）。
2. **「Create API key」**（APIキーを作成）というボタンを探して押します。
3. `AIza...` から始まる長いパスワードのような文字列が表示されたら、それを見失わないようにコピーします。これが「APIキー」です！
4. Kitchen App の **「設定」 > 「献立」** タブを開きます。
5. **「Gemini API（AI連携機能）の設定」** という項目の横にある「ロック中」ボタンを押してロックを解除します。
6. 先ほどコピーした長い文字列を貼り付けて、「保存」を押します。
7. 「接続テスト」ボタンを押して「✅ 接続成功」と表示されれば、設定完了です！AIがいつでもお手伝いできる状態になりました。

### よくある質問
* **Q. この設定はお金がかかるの？**
  * A. 基本的にすべて無料の範囲でご利用いただけます。AI（Gemini API）も通常のご家庭での利用範囲なら無料枠で十分にまかなえます。
* **Q. APIキーを誰かに知られても平気？**
  * A. **絶対に秘密にしてください。** 銀行の暗証番号と同じくらい大切なものです。Kitchen App はあなたのスマホの中（ローカル）にだけこのキーを保存し、こっそり外部に漏らすようなことは一切ありませんのでご安心ください。
