# Feature Guide

最終改訂: 2026-02-25

v1.9.7 時点の主要機能をユーザー視点で説明します。

---

## 1. ホーム

- Liquid GlassデザインのホームUI
- 旬のおすすめ（毎日ローテーション）
- 在庫一致率ベースのおすすめ表示
- 最新のWebP最適化カテゴリ画像タイルから検索遷移

---

## 2. レシピ検索

- Fuse.js によるあいまい検索
- カテゴリ / 機器 / 時短 / 旬フィルタ
- 在庫一致率表示
- 閲覧履歴・お気に入り・献立採用履歴を使った好み補正並び替え（Kitchen App Preference Rank）
- ヘルシオデリは下位表示

---

## 3. 在庫管理

- 食材の在庫ON/OFF + 数量管理
- 日本の家庭向け調味料プリセット（基本和食/定番追加など）の一括登録
- 表記ゆれに対応した一致率計算
- 50音順で一覧表示

---

## 4. 週間献立

- 7日分の主菜を自動選択
- 日ごとに副菜/スープを自動補完
- ロック / 差し替え / 再生成
- 複数レシピの調理スケジュール表示
- 調理開始/いただきます/ガント誘導アイコンを拡大した高視認タイル
- 主菜/副菜の人数調整ステッパー（押しやすい大型ボタン）
- Googleカレンダー登録時は、1日1イベントに主菜+副菜/スープをまとめて登録

### QRコードカレンダー共有（v1.9.7）

- 買い物リストをGoogleカレンダーに登録する際、週間献立のQRコード画像をイベントに添付
- 説明文にインポートURL（`?import-menu=...`）も埋め込み（テキスト共有との二重対応）
- 受信側はカレンダーQRをスキャンorURLタップで確認画面を経由してアプリへ取り込み可能
- QRデータはハイブリッド方式：公式レシピはIDのみ、カスタムレシピはフルデータ埋め込み

### 共有機能（v1.5.0）

- 共有リンク生成（`/weekly-menu?shared=...`）
- 共有コード貼り付けで献立取り込み

### 買い物リスト（v1.5.0改良）

- 初期提案は主材料中心（調味料はトグルで表示）
- 不足食材を追加・編集・削除できるUI
- LINE送信用コピー

## 7. AI機能

- URLからのレシピ解析（Gemini）
- 写真複数枚から食材抽出 -> 献立生成（2段階フロー）
- 取り込み結果は大型編集ウィンドウで食材名・分量・手順を編集して保存
- 週間献立の補助リファイン（任意）
- 設定タブでAPIキー登録・接続確認
- 機能別モデル選択（軽量/標準/高精度）
- AI推定使用量表示（今日の回数・モデル別回数・推定残り回数）
- Geminiチャットはタブ移動中も処理継続、履歴は約3日保持

### 代替食材の相談（v1.9.1）

レシピ詳細の材料行をタップして「不足」マークを付けると、
「代替食材を Gemini に相談」ボタンが材料セクション下部に出現。
タップするとレシピ全文脈（タイトル・調理器具・人数・全材料・不足食材）を含む
プロンプトが自動生成され、Gemini チャット画面の入力欄にセットされる。
内容を確認・編集してから手動で送信できる。

## 8. 設定画面

`/settings/:tab` のタブ遷移型UI:

- account
- calendar
- menu
- notify
- data
- guide (やりたいこと別のカード型ガイド)
- version

`version` タブでは、各リリースバージョンの変更要約と履歴を確認できます。
