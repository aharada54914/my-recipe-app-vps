# Feature Guide

最終改訂: 2026-02-21

v1.7.5 時点の主要機能をユーザー視点で説明します。

---

## 1. ホーム

- Liquid GlassデザインのホームUI
- 旬のおすすめ（毎日ローテーション）
- 在庫一致率ベースのおすすめ表示
- カテゴリ画像タイルから検索遷移

---

## 2. レシピ検索

- Fuse.js によるあいまい検索
- カテゴリ / 機器 / 時短 / 旬フィルタ
- 在庫一致率表示
- 閲覧履歴・お気に入り・献立採用履歴を使った好み補正並び替え（Kitchen App Preference Rank）
- ヘルシオデリは下位表示

---

## 3. 在庫管理

- 食材の在庫ON/OFF + 数量管理
- 表記ゆれに対応した一致率計算
- 50音順で一覧表示

---

## 4. 週間献立

- 7日分の主菜を自動選択
- 日ごとに副菜/スープを自動補完
- ロック / 差し替え / 再生成
- 複数レシピの調理スケジュール表示
- 調理開始/いただきます/ガント誘導アイコンを拡大した高視認タイル

### 共有機能（v1.5.0）

- 共有リンク生成（`/weekly-menu?shared=...`）
- 共有コード貼り付けで献立取り込み

### 買い物リスト（v1.5.0改良）

- 初期提案は主材料中心（調味料はトグルで表示）
- 不足食材を追加・編集・削除できるUI
- LINE送信用コピー

---

## 5. 通知（v1.5.0）

- 通知権限状態の表示と許可リクエスト
- 調理開始通知（指定時刻）
- 週間献立生成完了通知
- 買い物リスト確認通知

---

## 6. Google連携

- Googleログイン
- Google Driveバックアップ/復元
- Google Calendarイベント登録

---

## 7. AI機能

- URLからのレシピ解析（Gemini）
- 写真複数枚から食材抽出 -> 献立生成（2段階フロー）
- 取り込み結果は大型編集ウィンドウで食材名・分量・手順を編集して保存
- 週間献立の補助リファイン（任意）
- 設定タブでAPIキー登録・接続確認

---

## 8. 設定画面

`/settings/:tab` のタブ遷移型UI:

- account
- calendar
- menu
- notify
- data
- guide
- version

`version` タブでは、リリース間差分（V1以前→V1 / V1→V1.5.0 / V1.5.0→V1.6.0 / V1.6.0→V1.6.5 / V1.6.5→V1.7.0 / V1.7.0→V1.7.5）を確認できます。
