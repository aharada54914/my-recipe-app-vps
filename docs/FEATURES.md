# Feature Guide

最終改訂: 2026-03-07

v2.0.0 時点の主要機能を、現行 UI に合わせて要約します。

---

## 1. アプリ全体

- `Warm Tactile Kitchen` テーマ
  - light / dark / system を選択可能
  - コントラストと余白を優先したスマホ UI
- ラベル付き BottomNav
  - `ホーム / 献立 / Gemini / お気に入り / 履歴`
- sticky Header
  - 検索
  - 複数レシピスケジュール
  - 在庫管理
  - 設定 / 接続

---

## 2. ホーム

- 一次導線を `レシピ検索` と `AI に相談` に集約
- 二次導線として `今週の献立` サマリーを表示
- `今日食べたい料理`
  - 天気
  - 旬
  - 個人化スコア
  を合成して 4 件を表示
- `在庫でつくれるレシピ`
  - 在庫一致率上位を表示
- `旬のおすすめ`
  - 当月の旬食材と一致するレシピを表示
- `カテゴリ画像タイル`
  - カテゴリ検索へ遷移

---

## 3. レシピ検索

- Fuse.js によるあいまい検索
- カテゴリ / 機器 / 時短 / 旬フィルタ
  - 別グループは AND
  - 同一グループ内は OR
- カテゴリ、機種、条件はスマホで横スクロール不要
- active chip による個別解除
- `すべて解除`
- 好みシグナルによる並び替え補正
- 在庫一致率表示
- `最近の検索`
- iPhone Safari を意識した IME-safe 入力
  - 入力中は URL を即時更新しない
  - blur / submit で URL と履歴を同期

---

## 4. 在庫管理

- 食材検索から即時追加
- 明示的な操作
  - `+1`
  - `-1`
  - 数量直接入力
  - 削除
- `最近使った食材から追加`
- 日本の家庭向け調味料プリセット
- 在庫 QR 共有 / 受信

---

## 5. 週間献立

- 7 日分の主菜を自動生成
- 副菜 / スープ自動補完
- ロック、差し替え、再生成
- weather-aware な献立生成
- 画面構成をスマホ向けに圧縮
  - summary
  - 注目する 1 日
  - 残り日程
  - 買い物リスト
- 買い物リスト
  - 不足食材抽出
  - 追加 / 編集 / 削除
  - LINE 共有向けコピー
- 共有
  - 共有コード
  - 共有 URL
  - QR 取り込み
- Google Calendar 登録
  - 献立イベント
  - 買い物イベント

---

## 6. Google 連携

- Google ログイン
- Google Drive バックアップ / 復元
- Google Calendar 連携
- 接続状態を `StatusNotice` で一貫表示
- 状態に応じて `次にやること` が分かる導線を表示

### QA Google モード

- `?qa-google=1` で有効化
- 実アカウントなしで connected flow を検証可能
- `設定 > 詳細設定 > 接続フロー検証` から操作可能
- モック対象:
  - ログイン状態
  - Drive バックアップ / 復元
  - Calendar 登録

---

## 7. AI 機能

- URL からのレシピ解析
- 写真複数枚から食材抽出
- 在庫から提案
- Gemini チャット
- 保存前の大型編集ウィンドウ
- URL / 画像 / 提案失敗時の再試行導線
- API キー、モデル、接続状態、推定使用量の設定

---

## 8. 設定

`/settings/:tab`

- `account`
  - 表示名: `接続`
  - Google ログイン状態
  - Drive 連携状態
  - Calendar 登録先設定
- `planning`
  - 表示名: `献立`
  - 献立生成条件
  - 食事希望時刻
  - カレンダー登録時間帯
- `appearance`
  - `system / light / dark`
- `ai`
  - Gemini API キー
  - 機能別モデル
  - AI 推定使用量
- `notifications`
  - 通知権限と通知設定
- `data`
  - import / export
  - 在庫 QR 共有 / 受信
- `help`
  - `まずはここから / よく使う操作 / 共有・移行 / 困ったとき`
  - 状態チップ付きの記事
  - 各記事から設定 / 機能画面へ直接遷移
- `about`
  - 更新履歴
- `advanced`
  - OAuth 詳細設定
  - QA Google モード
