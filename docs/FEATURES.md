# Feature Guide

最終改訂: 2026-03-05

v2.0.0 時点の主要機能をユーザー視点で説明します。

---

## 1. ホーム（v2.0.0 全面刷新）

- Liquid GlassデザインのホームUI
- **今日食べたい料理**（2×2タイル）— Phase3スコアで選出した4件をWeeklyMenuTimeline直下に表示
  - Phase2ベクトルスコア40%: レシピ4D気象ベクトル（温料理/汁物/辛み/糖質）× 天気需要ベクトルのドット積（識別率約50%）
  - T_opt個人化スコア35%: 閲覧・献立採用履歴から学習した個人最適気温（T_opt）を用いた個人化スコア
  - 旬食材スコア25%: 食材が当月旬リストに含まれるか
  - 最終4件はSoftmax確率的サンプリング（temperature=0.4）で多様性を確保
  - 天気データ取得失敗時は旬食材フィルタのみのフォールバック
- **旬のおすすめ**（2×2タイル）— WeeklyMenuTimeline**直下**に配置。当月24種類プールから日替わりローテーション10件と一致するレシピを表示
- 在庫でつくれるレシピ（2×2タイル）— 在庫一致率上位4件を表示（在庫0件時は非表示）
- カテゴリ画像タイル（2×4グリッド）— WebP最適化画像からカテゴリ検索へ遷移
- WeeklyMenuTimeline（コンパクト表示）— 今日〜2日分の献立をヘッダー下に表示
- 検索バー下のクイックボタン（今日食べたい料理 / 在庫レシピ / 旬のレシピ）で各セクションへスムーズスクロール

### ヘッダー（v2.0.0）

- Packageアイコンボタン（在庫管理へ直接ジャンプ）をアカウントアイコンと設定アイコンの間に追加
- OAuth設定あり・未ログイン時にコンパクトな「ログイン」ボタンをヘッダーに表示

## 3. 在庫管理

## 4. 週間献立

- 7日分の主菜を自動選択
- 日ごとに副菜/スープを自動補完
- ロック / 差し替え / 再生成
- 複数レシピの調理スケジュール表示
- 調理開始/いただきます/ガント誘導アイコンを拡大した高視認タイル
- 主菜/副菜の人数調整ステッパー（押しやすい大型ボタン）
- Googleカレンダー登録時は、1日1イベントに主菜+副菜/スープをまとめて登録

### 天気パネル（v2.0.0）

- 折りたたみ式7日間天気パネルを週間献立ページ上部に追加（気象庁API取得）
- 各日に `WeatherIllustration`（湿度・降水量レイヤードイラスト）+ 最高/最低気温 + 降水目安を表示
- 手動「再取得」ボタンと最終取得日時を表示
- 取得した天気は `selectWeeklyMenu` に渡し、天気考慮の献立自動生成を実現
- 再生成時にエラーが発生した場合はアラートで通知（以前はサイレント失敗）
- 天気が6日しか表示されなかったバグを修正（重複 useEffect を削除し `filterForecastForWeek` を統一適用）

### QRコードカレンダー共有（v1.9.7）

- 買い物リストをGoogleカレンダーに登録する際、週間献立のQRコード画像をイベントに添付
- 説明文にインポートURL（`?import-menu=...`）も埋め込み（テキスト共有との二重対応）
- 受信側はカレンダーQRをスキャンorURLタップで確認画面を経由してアプリへ取り込み可能
- QRデータはハイブリッド方式：公式レシピはIDのみ、カスタムレシピはフルデータ埋め込み

### 共有機能（v1.5.0）

- 共有リンク生成（`/weekly-menu?shared=...`）
- 共有コード貼り付けで献立取り込み

### 買い物リスト（v1.5.0改良）

- 初期提案は主材料中心（調味料はトグルで表示）
- 不足食材を追加・編集・削除できるUI
- LINE送信用コピー

## 7. AI機能

- URLからのレシピ解析（Gemini）
- 写真複数枚から食材抽出 -> 献立生成（2段階フロー）
- 取り込み結果は大型編集ウィンドウで食材名・分量・手順を編集して保存
- 週間献立の補助リファイン（任意）
- 設定タブでAPIキー登録・接続確認
- 機能別モデル選択（軽量/標準/高精度）
- AI推定使用量表示（今日の回数・モデル別回数・推定残り回数）
- Geminiチャットはタブ移動中も処理継続、履歴は約3日保持

### 代替食材の相談（v1.9.1）

レシピ詳細の材料行をタップして「不足」マークを付けると、
「代替食材を Gemini に相談」ボタンが材料セクション下部に出現。
タップするとレシピ全文脈（タイトル・調理器具・人数・全材料・不足食材）を含む
プロンプトが自動生成され、Gemini チャット画面の入力欄にセットされる。
内容を確認・編集してから手動で送信できる。

## 8. 設定画面

`/settings/:tab` のタブ遷移型UI:

- account
- calendar
- menu
- notify
- data
- guide (やりたいこと別のカード型ガイド)
- version

`version` タブでは、各リリースバージョンの変更要約と履歴を確認できます。
