## Execution Mapping

### Automated suites
- `npm test`
- `npm run build`
- `npm --workspace apps/web run test:smoke:ci`
- `npm --workspace apps/web run workflow:validate`
- `npm --workspace apps/web run workflow:smoke`

### Live smoke
- VPS health: `bash scripts/ops/kitchenctl.sh health`
- Discord bot/API smoke via internal endpoints and runtime logs
- Google Calendar live registration and lookup
- Family calendar persistence check in Postgres

## Notes
- Gemini-backed live checks depend on quota and may degrade to API-level error-handling verification if quota is exhausted.
- Discord UI checks are validated through bot runtime, internal endpoints, and message side effects where possible.
