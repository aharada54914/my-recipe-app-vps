# サイトをインターネットに公開しよう！ Vercelかんたんデプロイガイド

## 1. はじめに

このガイドでは、君が作ったウェブサイトやアプリを、世界中の人が見られるようにインターネット上に公開する方法を説明するよ。今回は、**Vercel（バーセル）** という、とっても便利なサービスを使うんだ。

### Vercelってなに？

一言でいうと、「ウェブサイトを公開するためのロケット発射台」みたいなものだよ。君が書いたコード（設計図）をVercelに渡すと、あっという間にインターネットという宇宙に打ち上げて、みんなが見られるようにしてくれるんだ。

## 2. 準備するもの

ロケットを打ち上げる前に、いくつか準備が必要だよ。

- **GitHubアカウント**: 君のコード（設計図）を保管しておく場所。まだ持っていなかったら、おうちの人と相談して作ってみよう。
- **Vercelアカウント**: ロケット発射台を使うための利用登録。GitHubアカウントがあれば、すぐに登録できるよ。

## 3. いざ、打ち上げ！デプロイの手順

ここからは、実際に君のアプリをインターネットに公開する手順を見ていこう。「デプロイ」っていうのは、「公開する」って意味のかっこいい言い方だと思ってね。

### ステップ1：Vercelにログインしよう

1.  まず、[Vercelの公式サイト](https://vercel.com)にアクセスする。
2.  「Sign Up」や「Login」というボタンを探して、君の**GitHubアカウントでログイン**しよう。

### ステップ2：プロジェクトを追加しよう

1.  ログインすると、「Dashboard（ダッシュボード）」という管理画面が表示されるよ。
2.  「Add New...」→「Project」というボタンを探してクリックしよう。

### ステップ3：GitHubリポジトリと連携しよう

1.  「Import Git Repository」という画面が表示されるはずだ。
2.  ここに、君が作ったアプリのコードが保管されているGitHubリポジトリ（今回は `aharada54914/my-recipe-app`）が表示されるから、それを選んで「**Import**」ボタンを押そう。
    - もし表示されなかったら、VercelにGitHubリポジトリを見る権限を与える設定が必要かもしれない。画面の指示に従ってみよう。

### ステップ4：プロジェクトの設定を確認しよう

1.  リポジトリをインポートすると、プロジェクトの設定画面になるよ。
2.  Vercelはとても賢いから、君のプロジェクトがどんな技術（フレームワーク）でできているか自動で当ててくれるんだ（今回は「Vite」と表示されるはず）。
3.  基本的には、何も変更しなくて大丈夫！ そのまま下にある「**Deploy**」ボタンを押そう。

### ステップ5：打ち上げ完了を待とう！

「Deploy」ボタンを押すと、いよいよ打ち上げ開始だ！

Vercelが君のコードを読み込んで、インターネット上で動く形に組み立ててくれる（これを「ビルド」って言うよ）。

少し待っていると、画面に「**Congratulations!**」みたいなお祝いのメッセージと、花火のアニメーションが表示されるはず。これが出たら、打ち上げ成功だ！

### ステップ6：公開されたサイトを見てみよう！

成功画面に、君のサイトの**URL**（インターネット上の住所）が表示されているよ。例えば、`my-recipe-app-xxxx.vercel.app`みたいな感じだ。

このURLをクリックすれば、君が作ったアプリが、君のパソコンやスマホだけじゃなく、世界中のどこからでもアクセスできるようになったことを確認できるよ！

## 4. もっと便利に！自動デプロイ

Vercelのすごいところは、一度この設定をすれば、次からはもっと楽になることだ。

君がGitHubの`main`ブランチ（設計図の最新版）に新しいコードをプッシュ（更新）すると、Vercelがそれを自動で検知して、**全自動で新しいバージョンのサイトに更新してくれる**んだ。

つまり、一度ロケット発射台をセットすれば、あとは設計図を更新するだけで、勝手に最新のロケットを打ち上げてくれるってこと。すごく便利だろ？

これで君も、立派なウェブ開発者だ！どんどん新しい機能を作って、世界に公開していこう！
